# 📅 作业小龙·周学习计划指导

> 适用对象：一年级小学生
> 使用方式：家长根据此文档，在「家长模式 → 布置额外任务」中配置每日任务

---

## 一、每日固定任务（系统自动生成，无需配置）

每天系统会自动生成以下5项日常作业，孩子完成后获得经验和龙币：

| 科目 | 经验 | 龙币 | 特殊说明 |
|------|------|------|----------|
| 📐 数学 | +100 | +10 | 双倍经验！每天第一道数学还额外+20龙币（数学挑战赛） |
| 📝 语文 | +50 | +5 | - |
| 🔤 英语 | +50 | +5 | - |
| 📖 课外阅读 | +40 | +4 | - |
| 🏃 体育锻炼 | +30 | +3 | - |

---

## 二、建议周计划（家长用「额外任务」功能配置）

### 周一 ~ 周五（学习日）

#### 🌅 早晨（上学前 · 10分钟）
- [ ] 背诵古诗（1首）→ 配置：`背诵古诗` · +70经验 · +7龙币

#### 🌙 晚上（放学后 · 40-60分钟）

**周一：数学强化日**
1. 口算练习20题 → +80经验 · +8龙币
2. 数学练习册2页 → +100经验 · +10龙币
3. 整理书包 → +20经验 · +2龙币

**周二：语文日**
1. 抄写生字10个 → +60经验 · +6龙币
2. 读一篇课文 → +50经验 · +5龙币
3. 练字15分钟 → +40经验 · +4龙币

**周三：英语日**
1. 英语单词听写10个 → +60经验 · +6龙币
2. 读英语绘本 → +50经验 · +5龙币
3. 背诵乘法口诀 → +80经验 · +8龙币

**周四：复习日**
1. 口算练习20题 → +80经验 · +8龙币
2. 抄写生字10个 → +60经验 · +6龙币
3. 阅读课外书20分钟 → +40经验 · +4龙币

**周五：轻松日**
1. 读一篇课文 → +50经验 · +5龙币
2. 跳绳100个 → +30经验 · +3龙币
3. 整理书包 → +20经验 · +2龙币

---

### 周六 ~ 周日（周末）

#### 🎉 周末特殊活动
> **系统自动开启「周末双倍龙币」活动！** 所有龙币奖励 ×2

**周六：探索日**
1. 数学练习册2页 → +100经验 · +10龙币（双倍后+20龙币！）
2. 阅读课外书20分钟 → +40经验 · +4龙币
3. 跳绳100个 → +30经验 · +3龙币

**周日：复习 + 轻松**
1. 背诵乘法口诀 → +80经验 · +8龙币
2. 英语单词听写10个 → +60经验 · +6龙币
3. 自由阅读（30分钟以上）→ +40经验 · +4龙币

## 三、每日行为评价参考（家长端操作）

在「家长模式 → 行为评价」中，根据孩子当天表现进行评价：

### ✅ 建议每日关注的正向行为
| 行为 | 龙币 | 备注 |
|------|------|------|
| 📖 主动阅读 | +10 | 不用催就看书的奖励 |
| ✏️ 认真完成作业 | +15 | 作业质量高的奖励 |
| 📝 作业工整 | +10 | 书写整齐 |
| 👟 主动运动 | +10 | 自发锻炼 |
| 🌅 早睡早起 | +10 | 21点前睡觉 |

### ⚠️ 需要纠正的行为
| 行为 | 龙币 | 备注 |
|------|------|------|
| 😤 发脾气 | -5 | 适度扣分，重在引导 |
| 📱 电子产品超时 | -10 | 超过约定时间 |
| 🗑️ 乱丢垃圾 | -5 | 环保意识 |

## 六、家长操作速查

| 操作 | 位置 | 频率 |
|------|------|------|

---

> 💡 **小贴士**：系统有「每日衰减」机制——小龙的饱腹度、心情、亲密度每天会自然下降。如果孩子2-3天不互动，小龙会变得不开心。这可以用来引导孩子每天养成学习习惯！

> 🎁 **彩蛋**：孩子每次打开页面有20%概率收到小龙的随机惊喜礼物！周末所有龙币奖励自动翻倍！
